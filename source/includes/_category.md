# Category

## Get List Category


```shell
curl -X GET \
  https://shoupbud.xyz/api/category \
  -H 'accept: application/json' \
  -H 'cache-control: no-cache'
```

> Response Success (200)

```json
[
  {
    "id": 1,
    "name": "Uncategorized",
    "slug": "uncategorized",
    "description": null
  }
]
```
